== Description ==

Server & Website Info is a powerful yet easy-to-use WordPress plugin that displays detailed information about your server configuration, WordPress installation, and database settings. All information is presented in a clean, modern interface with a card-based layout.

= Features =

* **Server Information:** PHP version, server software, IP, hostname, MySQL version, system uptime, and memory limit
* **WordPress Information:** WordPress version, active theme, debug mode, permalink structure, and post revisions
* **Database Information:** Database name, user, host, charset, and collation
* **Modern Interface:** Clean and responsive card-based layout
* **Click to Copy:** Easily copy any value to clipboard with a single click
* **Multilingual:** Supports multiple languages including English (US/UK), French, German, Spanish, Chinese, Japanese, Arabic, and Turkish

= Usage =

1. Install and activate the plugin
2. Go to "Server & Website Info" in your WordPress admin menu
3. View comprehensive information about your WordPress installation
4. Click any value to copy it to clipboard

= Supported Languages =

Don't see your language? Help us translate the plugin into your language!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/server-website-info` directory, or install the plugin through the WordPress plugins screen directly
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the "Server & Website Info" menu item to view your system information

== Frequently Asked Questions ==

= What information does this plugin show? =

The plugin displays comprehensive information about:
* Server configuration (PHP version, server software, etc.)
* WordPress installation details
* Database settings

= Can I copy the information? =

Yes! Simply click on any value to copy it to your clipboard. A tooltip will confirm when the text has been copied.

= Is this plugin compatible with my WordPress version? =

The plugin is compatible with WordPress 5.0 and above. We regularly test it with the latest WordPress version.

= Does this plugin work with multisite? =

Yes, the plugin is fully compatible with WordPress multisite installations.
